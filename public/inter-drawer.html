Keyboard Shortcuts Reference

Enable Shortcuts: Press ` (backtick) when a GeoView element is focused to toggle keyboard shortcuts on/off

When shortcuts are enabled:

  • D - Toggle Drawing mode
  • E - Toggle Editing mode
  • G - Cycle geometry type forward
  • Shift+G - Cycle geometry type backward
  • S - Open style menu
  • M - Toggle measurements
  • N - Toggle snapping
  • Shift+C - Clear all drawings
  • Shift+S - Save / Download drawings
  • Shift+O - Open / Upload drawings

Always available within GeoView Element:

  • Ctrl+Z - Undo
  • Ctrl+Y or Ctrl+Shift+Z - Redo
  • Escape - Clear selection/exit editing

Crosshair mode (Ctrl+Alt+M):

  • Enter or Space - Add drawing vertex / Select feature for editing
  • Shift+Enter or Shift+Space - Finish drawing (While Drawing) or Delete vertex / Open text editor (While Editing)
  • Escape - Cancel grabbed handle
  • Ctrl+↑ - Zoom in
  • Ctrl+↓ - Zoom out
  • Shift+↑ - Increase pan delta
  • Shift+↓ - Decrease pan delta
