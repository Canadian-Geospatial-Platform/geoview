How to Use This Template

Replace the UUID in the URL path to dynamically load different layers from the GeoCore catalog. Simply modify the URL structure: /demo-open-maps/{UUID}

Example: http://localhost:8080/demo-open-maps/6c343726-1e92-451a-876a-76e17d398a1c

You can also load multiple layers by separating UUIDs with commas: /demo-open-maps/{UUID1},{UUID2}
