- Added `packages/react-jsx-global.d.ts` — global JSX namespace shim (React 19 removed the global JSX namespace from @types/react)

**Code fixes for React 19 breaking changes:**
- All `useRef<T>()` calls (no argument) updated to `useRef<T>(undefined)` or `useRef<T>(null)` — React 19 requires an initial value
- All `RefObject<T>` interface props updated to `RefObject<T | null>` — React 19's `useRef<T>(null)` returns `RefObject<T | null>` instead of `MutableRefObject<T | null>`
- Removed `React.MutableRefObject` usage (removed in @types/react 19)
- Removed `MutableRefObject` from `external-types.ts` re-exports
- Fixed `FocusTrap` children type — `ReactElement` defaults to `ReactElement<unknown>` in React 19 (was `ReactElement<any>` in React 18)
- Fixed `cloneElement` type — requires explicit props type parameter in React 19
- Fixed `react-draggable` findDOMNode removal — added `nodeRef` prop to `<Draggable>` in swiper.tsx
- Replaced `@mui/base` imports: `FocusTrap` from `@mui/material/Unstable_TrapFocus`, `Mark` type from `@mui/material/Slider/useSlider.types`
- panel.tsx: changed `panelRef` from `HTMLButtonElement` to `HTMLDivElement` (Card renders a div)
